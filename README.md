## ‚öôÔ∏è | How To Set Up

**ALL** of these steps require the ***`SERDiscordConfig.txt`*** file.


![](https://i.postimg.cc/cLSbkJvw/Screenshot-2026-01-30-172024.png)

1. Set your server name.
    - Change the value of the `$serverName` variable to be your server name.
    E.g. `$serverName = "My Cool Server"`

2. Create 2 webhooks, one for the server list and one for the server logs.
